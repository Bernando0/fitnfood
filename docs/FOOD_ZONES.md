# Зоны еды 🟢🟡🔴⚪ — на чём основано

Бот относит каждый приём пищи в цветовую зону. Система не выдумана — она собрана из трёх
признанных подходов к оценке еды:

## 1. Noom — цветовая система по калорийной плотности
Noom делит еду на 🟢/🟡/🟠 по **калорийной плотности** = ккал ÷ граммы. Зелёное — наименее
плотное (овощи, фрукты, цельные крупы, нежирный белок), оранжевое — самое плотное (масла,
фастфуд, сладости). Идея: на зелёном наедаешься меньшими калориями.
Источник: [How Noom's Food Color System Works](https://www.noom.com/support/faqs/question-topics/food-logging/2015/12/what-do-the-green-yellow-and-red-food-color-categories-mean/)

> Важно: Noom намеренно **не запрещает** красное и не стыдит. Наш бот по твоему запросу
> жёстче по тону — но критикует ЕДУ И ВЫБОР, а не человека.

## 2. UK FSA — «светофор» по нутриентам на 100 г
Британский front-of-pack стандарт. Пороги на 100 г продукта:

| Нутриент | 🟢 Low | 🟡 Medium | 🔴 High |
|---|---|---|---|
| Жир | ≤ 3.0 г | 3.1–17.5 г | > 17.5 г |
| Насыщенный жир | ≤ 1.5 г | 1.6–5.0 г | > 5.0 г |
| Сахар | ≤ 5.0 г | 5.1–22.5 г | > 22.5 г |
| Соль | ≤ 0.3 г | 0.3–1.5 г | > 1.5 г |

Источник: [UK Traffic Light Food Labels](https://www.menusano.com/uk-traffic-light-food-labelling-system/),
[FSA/abdn guide (PDF)](https://www.abdn.ac.uk/media/site/rowett/documents/Traffic_light_guide.pdf)

## 3. NOVA — степень переработки
Группы 1–4. Группа 4 (**ультрапереработанное**: газировка, снеки, фастфуд, готовые блюда с
5+ промышленными ингредиентами) — сильный сигнал в красную зону.
Источник: [NOVA classification (Britannica)](https://www.britannica.com/topic/Nova-scale),
[Ultra-Processed Foods (PMC)](https://pmc.ncbi.nlm.nih.gov/articles/PMC6389637/)

## Как бот это применяет

**Цвет одного приёма (`health_score`)** ставит зрение-модель по объединённому правилу
(см. `bot/llm/prompts.py`): минимальная обработка + низкая плотность + постный белок/овощи →
🟢; калорийно-плотное/рафинированное → 🟡; жареное/ультрапереработанное/сладкие напитки/
фастфуд → 🔴. При сомнении — более строгая зона.

**Цвет дня** (`bot/services/zones.py`) — агрегат всех приёмов за день, **взвешенный по
калориям**: 🟢=+1, 🟡=0, 🔴=−1; среднее > 0.34 → 🟢, < −0.34 → 🔴, иначе 🟡. Большой красный
приём перевешивает маленький зелёный. **Нет ни одного фото за день → ⚪ серая зона.**

**Калорийные ориентиры** (реальные порции), зашитые в промпт, чтобы не занижать:
донер/шаурма 600–1100 • бургер 500–900 • пицца кусок 280–400 • фри 300–500 •
фраппучино/латте с сиропом 300–500 • суши-сет 500–900 • плов 500–800.
